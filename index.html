Where to find this "thumbprint"?

  1. On core server go to mmc
  2. Add/remove snapin
  3. Certificates
  4. Add
  5. Computer - local accounts
  6. In the certificates click on certificate that is named as current server
  7. Double click on it and go to details
  8. Click on thumbprint, you can copy it from here
